## HOM-01 · 간부 홈 (기본)

**진입 경로:** 로그인 성공 (간부) / 바텀 네비 > 홈 탭

---

### 레이아웃

```
┌─────────────────────────────┐
│  아차사고 관리          🔔 ☰ │  ← 상단바
├─────────────────────────────┤
│                             │
│  ┌─────────────────────┐   │
│  │  내 사고 현황        │   │  ← 요약 카드
│  │                     │   │
│  │  전체   조치완료  대기│   │
│  │   12      9       3 │   │
│  └─────────────────────┘   │
│                             │
│  최근 등록 사고              │  ← 섹션 라벨
│                             │
│  ┌─────────────────────┐   │
│  │ ⚡ 감전   조치완료   │   │  ← 최근 사고 카드 1

│  │ 2026.05.14          │   │
│  └─────────────────────┘   │

│  │  📋    │ │  📁    │ │  🔍    │ │
│  │사고등록│ │내사고  │ │유사사례│ │
│  └────────┘ └────────┘ └────────┘ │
│                             │
│  ── 통계 그래프 ────────── │  ← 섹션 라벨
│                             │
│  [ 월별/유형별 분석 현황 ]   │
│  (꺾은선 그래프)             │
│                             │
│  [ 예하부대 등록 현황 ]      │
│  (스택 막대그래프)           │
│                             │
├─────────────────────────────┤
│  🏠홈  📋등록  📁관리  📢공지  👤내정보 │  ← 바텀 네비
└─────────────────────────────┘
```

---

### 컴포넌트 상세

#### 상단바
- 좌: 앱 타이틀 "아차사고 관리"
- 우: 벨 아이콘 (🔔) — 읽지 않은 알림 시 뱃지 표시 / 탭 시 알림 목록
- 우: 햄버거 메뉴 (☰) — 간부는 비활성 (통계 그래프는 홈 하단에서 직접 확인)
  - ※ 관리자·지휘관만 햄버거로 통계 진입, 간부는 해당 없음

#### 내 사고 현황 요약 카드
- 전체 등록 건수 / 조치 완료 건수 / 조치 대기 건수 (3열)
- 조치 대기 건수: 강조 색상 (주황 계열)
- 카드 전체 탭: 내 사고 관리 목록으로 이동

#### 최근 등록 사고
- 최대 2건 표시
- 항목 구성:
  - 사고 유형 아이콘 + 유형명
  - 처리 상태 뱃지 (조치 완료 / 조치 대기)
  - 발생 장소
  - 등록 일자
  - 최근 수정 시간 (수정 이력 있을 때만 표시, 예: "5분 전 수정")
- 항목 탭: MGT-02 (사고 상세)로 이동
- 목록이 비어있을 경우: "아직 등록된 사고가 없어요" 빈 상태 표시

#### 빠른 메뉴
- [사고 등록]: REG-01로 이동
- [내 사고 관리]: MGT-01로 이동
- [유사 사례]: REG-10으로 이동 (독립 진입 — 유형 필터 자동 적용 없이 전체 목록 표시)
- 3열 그리드, 아이콘 + 텍스트 라벨

#### 통계 그래프
- ① 월별/유형별 아차사고 분석 현황
  - 꺾은선 그래프 (월별 추이, 유형별 범례)
- ② 예하부대 아차사고 등록 현황
  - 스택 막대그래프 (조치완료/조치대기)
  - 기간 선택 탭: 이번달 / 3개월 / 6개월 / 1년
  - 카테고리 필터 칩: 전체 + 10개 유형
- 그래프 영역 탭: 확대 보기 (풀스크린 모달)

#### 바텀 네비게이션
- 홈 / 사고 등록 / 내 사고 관리 / 공지사항 / 내 정보 (5탭)
- 현재 탭 강조

---

### 상태별 UI

| 상태 | UI 처리 |
|---|---|
| 최초 진입 (등록 건 없음) | 최근 사고 빈 상태 메시지 표시 |
| 조치 대기 건 있음 | 요약 카드 조치 대기 수치 강조 |
| 읽지 않은 알림 있음 | 벨 아이콘에 숫자 뱃지 |

## HOM-02 · 간부 홈 (조치 기한 임박 배너)

**진입 경로:** HOM-01과 동일 — 조치 대기 건 존재 + 기한 3일 이내 조건 충족 시

---

### 레이아웃

```
┌─────────────────────────────┐
│  아차사고 관리          🔔 ☰ │
├─────────────────────────────┤
│ ┌─────────────────────────┐ │
│ │ ⚠ 조치 기한이 임박했어요│ │  ← 상단 배너
│ │ 미완료 보고서가 있어요.  │ │
│ │ 조치를 완료해주세요.    │ │
│ │ [지금 완료하기]  [나중에]│ │
│ └─────────────────────────┘ │
│                             │
│  (이하 HOM-01과 동일)        │
│                             │
├─────────────────────────────┤
│  🏠홈  📋등록  📁관리  📢공지  👤내정보 │
└─────────────────────────────┘
```

---

### 컴포넌트 상세

#### 조치 기한 임박 배너
- 표시 조건: 조치 대기 건 존재 + 기한 3일 이내
- 배경: 경고색 (주황/빨강 계열)
- 텍스트: "미완료 보고서가 있어요. 조치를 완료해주세요."
- [지금 완료하기] 버튼: 가장 기한이 촉박한 사고 상세(MGT-02)로 이동
- [나중에] 버튼: 배너 닫기 (세션 내에서만 닫힘, 재진입 시 다시 표시)
- 미표시 조건: 기한 여유 있음 / 조치 대기 건 없음

### 컴포넌트 상세

#### 헤더
- 좌: 뒤로가기 (←)
- 중: "내 사고 관리"

#### 필터 탭
- 전체 / 조치 대기 / 조치 완료 (3탭)
- 각 탭에 건수 표시 (예: "대기 3")
- 탭 전환 시 목록 즉시 필터링

#### 사고 항목 카드
- 사고 유형 아이콘 + 유형명 (좌상단)
- 발생 장소 (본문)
- 등록 일시 (본문)
- 처리 상태 뱃지 (우하단)
  - ✅ 조치 완료: 초록
  - 🕐 조치 대기: 주황
- 최근 수정 시간 (수정 이력 있을 때만, 예: "5분 전 수정") — 우하단 작은 텍스트
- 카드 탭: MGT-02 (사고 상세)로 이동
- 정렬: 등록일 내림차순 (최신 순)

#### 빈 상태
- 해당 필터에 결과 없음: "해당하는 사고가 없어요" + 일러스트

## MGT-02 · 사고 상세

**진입 경로:** MGT-01 항목 탭 / HOM-01 최근 사고 카드 탭 / HOM-02 배너 "지금 완료하기"

### 레이아웃

```
┌─────────────────────────────┐

├─────────────────────────────┤
│                             │
│  ┌──────┬──────┬──────┐    │
│  │ 등록  │ AI생성│ 조치  │   │  ← 사진 영역 (가로 스크롤)
│  │ 사진  │  사진 │ 완료  │   │
│  └──────┴──────┴──────┘    │
│                             │
│  ┌──────────────────────┐  │
│  │  🕐 조치 대기         │  │  ← 처리 상태 뱃지
│  └──────────────────────┘  │
│                             │
│  사고 유형                   │
│  ⚡ 감전                     │
│                             │

│  ● 높음                      │
│                             │

│  2026년 05월 15일 14:30      │
│                             │
│  발생 장소                   │
│  3중대 창고 앞               │
│                             │
│  식별된 위험 요인             │
│  전선이 노출된 채로 방치되어   │
│  접촉 위험이 있었음           │
│                             │
│  조치 결과                   │
│  미작성 (조치 대기)           │
│                             │
│  ─────────────────────────  │

├─────────────────────────────┤

└─────────────────────────────┘
```

---

### 컴포넌트 상세

#### 헤더
- 좌: 뒤로가기 (←)
- 중: "사고 상세"
- 우: [수정] 버튼 — 항상 활성
  - 탭 시: MGT-03으로 이동

#### 사진 영역
- 가로 스크롤 (페이징 또는 자유 스크롤)
- 등록 사진 → AI 생성 사진 → 조치 완료 사진 순서
- 사진 없는 슬롯: "사진 없음" 텍스트 표시
- 사진 탭: 풀스크린 뷰어 (핀치 줌 지원)
- 사진 유형 라벨 표시 (등록 / AI생성 / 조치완료)

#### 처리 상태 뱃지
- ✅ 조치 완료: 초록 배경
- 🕐 조치 대기: 주황 배경

#### 정보 항목 (읽기 전용)
| 항목 | 내용 |
|---|---|
| 사고 유형 | 유형 아이콘 + 유형명 |
| 위험도 | AI 자동 산출 뱃지 + 수준 표시 (낮음/보통/높음/매우 높음) |
| 발생 일시 | YYYY년 MM월 DD일 HH:MM |
| 발생 장소 | 텍스트 |
| 식별된 위험 요인 | 멀티라인 텍스트 |
| 조치 결과 | 텍스트 / 조치 대기 시 "미작성 (조치 대기)" |

#### 수정 이력 영역
- "최근 수정: N분 전" — 수정 이력 있을 때만 표시
- "수정 이력 보기 [>]" — 탭 시 MGT-04로 이동

#### 하단 버튼
- [조치 완료 처리] — Primary, full-width
  - 표시 조건: 조치 대기 상태일 때만
  - 탭 시: MGT-05 (조치 완료 인증)으로 이동
- 조치 완료 상태: 하단 버튼 없음

```
│                             │

│                             │
│  식별된 위험 요인             │
│  ┌──────────────────────┐  │
│  │ 전선이 노출된 채로...  │  │  ← 멀티라인 입력
│  └──────────────────────┘  │
│                             │
│  조치 결과                   │
│  ┌──────────────────────┐  │
│  │ 안전장비 착용 및...    │  │  ← 멀티라인 입력
│  └──────────────────────┘  │
│                             │
└─────────────────────────────┘
```

---

### 컴포넌트 상세

#### 헤더
- 좌: [취소] — 탭 시 변경 취소 확인 다이얼로그
- 중: "사고 수정"
- 우: [저장] — 탭 시 서버 전송 + MGT-02 복귀

#### 취소 확인 다이얼로그
```
변경 중인 내용이 있어요.
수정을 취소할까요?

[계속 수정]  [취소하기]
```

#### 수정 가능 항목
| 항목 | 수정 방식 |
|---|---|
| 사진 | 추가(카메라/갤러리) / 삭제(썸네일 탭 후 삭제 버튼) |
| 식별 일시 | 날짜+시간 피커 바텀시트 |
| 발생 장소 | 단일 라인 텍스트 필드 |
| 사고 유형 | 유형 선택 모달 (REG-04와 동일) |
| 식별된 위험 요인 | 멀티라인 텍스트 |
| 조치 결과 | 멀티라인 텍스트 |

#### 수정 불가 항목
- AI 생성 사진: 표시는 되나 편집 불가 ("AI 생성" 라벨 + 잠금 표시)
- 위험도: 표시 없음 (AI 자동 산출이므로 수정 화면에서 노출 불필요)

#### 저장 처리
- 서버 전송 성공: "수정되었어요" 토스트 + MGT-02로 이동
- 서버 전송 실패: "저장에 실패했어요. 다시 시도해주세요." 토스트

### 컴포넌트 상세

#### 헤더
- 좌: 뒤로가기 (←) — 탭 시 MGT-02로 복귀 (사진 첨부 취소)
- 중: "조치 완료 확인"

#### 사진 섹션 (3개 구역)
| 구역 | 내용 | 수정 |
|---|---|---|
| 등록 사진 | 1단계 첨부 사진 | 불가 (읽기 전용) |
| AI 생성 사진 | 3단계 AI 생성 이미지 | 불가 (읽기 전용) |
| 조치 완료 사진 | 방금 촬영/선택한 사진 | 추가 가능 |

- 각 섹션: 썸네일 가로 스크롤
- 사진 없는 섹션: "사진 없음" 표시
- [📷 사진 추가]: 조치 완료 사진만 추가 가능 (카메라/갤러리 바텀시트)

#### 하단 버튼
- [조치 완료 등록] — Primary, full-width
- 탭 시: 서버 전송
  - 성공: "조치가 완료되었어요" 토스트 + MGT-01으로 이동 (상태 갱신)
  - 실패: "등록에 실패했어요. 다시 시도해주세요." 토스트

## 화면 간 이동 흐름 요약

```
HOM-01 (간부 홈)
  ├─ 요약 카드 탭 ──────────────────────── MGT-01 (목록)
  ├─ 최근 사고 카드 탭 ─────────────────── MGT-02 (상세)
  ├─ 빠른 메뉴 > 내 사고 관리 ──────────── MGT-01 (목록)
  ├─ 빠른 메뉴 > 사고 등록 ─────────────── REG-01
  └─ 빠른 메뉴 > 유사 사례 ─────────────── REG-10 (전체 목록)

HOM-02 (배너 상태)
  └─ 지금 완료하기 ─────────────────────── MGT-02 (가장 기한 촉박 사고)

MGT-01 (목록)
  └─ 항목 탭 ──────────────────────────── MGT-02 (상세)

MGT-02 (상세)
  ├─ [수정] ────────────────────────────── MGT-03 (수정)
  ├─ 수정 이력 보기 ────────────────────── MGT-04 (이력)
  └─ [조치 완료 처리] ──────────────────── MGT-05 (인증 확인)

MGT-03 (수정)
  └─ [저장] ────────────────────────────── MGT-02 (상세, 갱신)

MGT-05 (인증 확인)
  └─ [조치 완료 등록] ──────────────────── MGT-01 (목록, 상태 갱신)
```
