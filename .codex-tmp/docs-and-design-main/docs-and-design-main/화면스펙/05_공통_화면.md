<a name="cmn-01"></a>
## CMN-01 · 로그인

**진입 경로:** 앱 최초 실행 / 로그아웃 후 / 세션 만료

---

### 레이아웃

```
┌─────────────────────────────┐
│                             │
│                             │
│       [ 앱 로고 ]            │
│     아차사고 관리 시스템      │
│                             │
│                             │
│  아이디                      │
│  ┌──────────────────────┐  │
│  │ 아이디를 입력해주세요 │  │
│  └──────────────────────┘  │
│                             │
│  비밀번호                    │
│  ┌──────────────────────┐  │
│  │ 비밀번호를 입력해주세요│  │
│  └──────────────────────┘  │
│                             │
│  [ 로그인 ]                 │
│                             │
│  계정이 없으신가요?          │
│  회원가입                    │  ← 텍스트 버튼
│                             │
└─────────────────────────────┘
```

---

### 컴포넌트 상세

#### 앱 로고 + 타이틀
- 앱 아이콘 + "아차사고 관리 시스템" 텍스트
- 상단 여백 충분히 확보 (로고 중앙 배치)

#### 아이디 필드
- 라벨: "아이디"
- 입력: 단일 라인 텍스트 필드
- Placeholder: "아이디를 입력해주세요"
- 키보드: 영숫자

#### 비밀번호 필드
- 라벨: "비밀번호"
- 입력: 비밀번호 필드 (마스킹)
- Placeholder: "비밀번호를 입력해주세요"
- 우측 눈 아이콘: 탭 시 비밀번호 표시/숨김 전환

#### 로그인 버튼
- Primary, full-width
- **비활성 조건:** 아이디 또는 비밀번호 비어있음
- **활성 조건:** 둘 다 1자 이상 입력
- 탭 시: 서버 인증 요청

#### 로그인 처리 결과
| 결과 | 처리 |
|---|---|
| 성공 | 권한 확인 후 해당 홈으로 이동 |
| 승인 대기 중 | CMN-03으로 이동 |
| 실패 (아이디/비밀번호 불일치) | "아이디 또는 비밀번호가 올바르지 않아요" 토스트 |
| 네트워크 오류 | "네트워크 연결을 확인해주세요" 토스트 |

#### 회원가입 버튼
- 텍스트 버튼 (secondary)
- 탭 시: CMN-02로 이동

<a name="cmn-02"></a>
## CMN-02 · 회원가입

**진입 경로:** CMN-01 > "회원가입" 버튼 탭

**다음 화면:** CMN-03 (가입 신청 성공 시)

---

### 레이아웃

```
┌─────────────────────────────┐
│ [←]   회원가입              │
├─────────────────────────────┤
│                             │
│  계급                        │
│  ┌──────────────────────┐  │
│  │ 계급 선택         [∨]│  │  ← 드롭다운
│  └──────────────────────┘  │
│                             │
│  이름                        │
│  ┌──────────────────────┐  │
│  │ 홍길동               │  │
│  └──────────────────────┘  │
│                             │
│  군번                        │
│  ┌──────────────────────┐  │
│  │ 23-123456            │  │
│  └──────────────────────┘  │
│                             │
│  소속 부대                   │
│  ┌──────────────────────┐  │
│  │ ○○부대 1중대         │  │
│  └──────────────────────┘  │
│                             │
│  아이디                      │
│  ┌──────────────────────┐  │
│  │ hong123              │  │
│  └──────────────────────┘  │
│                             │
│  비밀번호                    │
│  ┌──────────────────────┐  │
│  │ ••••••••             │  │
│  └──────────────────────┘  │
│                             │
│  비밀번호 확인               │
│  ┌──────────────────────┐  │
│  │ ••••••••             │  │
│  └──────────────────────┘  │
│                             │
├─────────────────────────────┤
│  [ 가입 신청 ]              │
└─────────────────────────────┘
```

---

### 컴포넌트 상세

#### 헤더
- 좌: 뒤로가기 (←) — CMN-01으로 복귀
- 중: "회원가입"

#### 입력 필드

| 필드 | 타입 | 필수 | 비고 |
|---|---|---|---|
| 계급 | 드롭다운 | ✓ | 이병~준위 등 계급 목록 |
| 이름 | 텍스트 | ✓ | 한글 |
| 군번 | 텍스트 | ✓ | 형식: YY-XXXXXX |
| 소속 부대 | 텍스트 | ✓ | 부대명 + 중대 |
| 아이디 | 텍스트 | ✓ | 영숫자 조합 |
| 비밀번호 | 비밀번호 | ✓ | 8자 이상 |
| 비밀번호 확인 | 비밀번호 | ✓ | 위와 일치 여부 실시간 확인 |

#### 실시간 유효성 검사
- 비밀번호 확인: 입력 중 일치 여부 아이콘(✓/✗) 표시
- 비밀번호: 8자 미만 시 "8자 이상 입력해주세요" 인라인 안내

#### 가입 신청 버튼
- Primary, full-width
- **비활성 조건:** 필수 항목 미입력 / 비밀번호 불일치
- **활성 조건:** 전 항목 유효
- 탭 시: 서버 전송

#### 처리 결과
| 결과 | 처리 |
|---|---|
| 성공 | CMN-03으로 이동 |
| 중복 아이디 | "이미 사용 중인 아이디예요" 인라인 오류 표시 |
| 기타 실패 | "가입 신청에 실패했어요" 토스트 |

<a name="cmn-03"></a>
## CMN-03 · 승인 대기

**진입 경로:** CMN-02 가입 신청 성공 / CMN-01 로그인 (승인 대기 상태)

---

### 레이아웃

```
┌─────────────────────────────┐
│                             │
│                             │
│          ⏳                  │  ← 대기 아이콘
│                             │
│    관리자 승인 대기 중이에요  │
│                             │
│  가입 신청이 완료되었어요.    │
│  관리자가 계정을 승인하면     │
│  앱을 사용할 수 있어요.      │
│                             │
│  승인 완료 시 알림으로       │
│  안내드릴게요.               │
│                             │
│                             │
│                             │
├─────────────────────────────┤
│  [ 로그인 화면으로 이동 ]    │
└─────────────────────────────┘
```

---

### 컴포넌트 상세

- 대기 아이콘: 모래시계 또는 시계 일러스트
- 메인 텍스트: "관리자 승인 대기 중이에요"
- 서브 텍스트: 가입 완료 안내 + 알림 안내
- [로그인 화면으로 이동]: CMN-01으로 이동

#### 승인 거절 상태 (거절 푸시 알림 수신 후 앱 재진입 시)
```
│          ✗                  │
│                             │
│    가입이 거절되었어요       │
│                             │
│  사유: 소속 부대 확인 불가   │  ← 거절 사유 (있을 경우)
│                             │
│  [ 로그인 화면으로 이동 ]    │
```

<a name="cmn-04"></a>
## CMN-04 · 알림 목록

**진입 경로:** 상단바 벨 아이콘 탭 (전 권한 공통)

---

### 레이아웃

```
┌─────────────────────────────┐
│ [←]   알림                  │
├─────────────────────────────┤
│                             │
│  오늘                        │  ← 날짜 구분선
│                             │
│  ┌─────────────────────┐   │
│  │ 🔴 미조치 기한 초과  │   │  ← 알림 항목 (읽지 않음)
│  │ [감전] 사고 조치 기한│   │
│  │ 이 초과되었어요.     │   │
│  │ 방금               │   │
│  └─────────────────────┘   │
│                             │
│  ┌─────────────────────┐   │
│  │ ✅ 가입 승인 완료    │   │  ← 읽은 알림 (회색)
│  │ 홍길동 하사 계정이   │   │
│  │ 승인되었어요.        │   │
│  │ 2시간 전            │   │
│  └─────────────────────┘   │
│                             │
│  어제                        │  ← 날짜 구분선
│                             │
│  ┌─────────────────────┐   │
│  │ 📢 공지사항          │   │
│  │ [안전 교육] 5월 정기  │   │
│  │ 교육 일정 안내       │   │
│  │ 어제                │   │
│  └─────────────────────┘   │
│                             │
│  ... (스크롤)               │
│                             │
└─────────────────────────────┘
```

---

### 컴포넌트 상세

#### 헤더
- 좌: 뒤로가기 (←)
- 중: "알림"
- 우: "전체 읽음" 텍스트 버튼 (읽지 않은 알림 있을 때만 표시)

#### 날짜 구분선
- 오늘 / 어제 / YYYY.MM.DD 형식

#### 알림 항목
- 읽지 않음: 일반 배경 + 좌측 색상 도트 또는 굵은 텍스트
- 읽음: 회색 처리
- 구성:
  - 알림 유형 아이콘 + 유형 라벨
  - 알림 내용 (2줄 말줄임)
  - 수신 시간 (방금 / N분 전 / N시간 전 / 어제 / 날짜)
- 항목 탭: CMN-05로 이동 + 읽음 처리

#### 알림 유형별 아이콘
| 유형 | 아이콘 | 색상 |
|---|---|---|
| 미조치 기한 초과 | ⚠ | 빨강 |
| 조치 촉구 | 🔔 | 주황 |
| 가입 승인 | ✅ | 초록 |
| 가입 거절 | ❌ | 빨강 |
| 공지사항 | 📢 | 파랑 |
| 반려 | 🔄 | 주황 |

#### 빈 상태
- "새로운 알림이 없어요"

<a name="cmn-05"></a>
## CMN-05 · 알림 상세

**진입 경로:** CMN-04 항목 탭

---

### 레이아웃

```
┌─────────────────────────────┐
│ [←]   알림 상세             │
├─────────────────────────────┤
│                             │
│  ⚠ 미조치 기한 초과         │  ← 알림 유형
│  2026.05.16 14:32           │  ← 수신 시간
│                             │
│  ─────────────────────────  │
│                             │
│  [감전] 사고 조치 기한이     │
│  초과되었어요.               │
│                             │
│  3중대 창고 앞에서 발생한    │
│  감전 사고의 조치 기한이     │
│  지났어요. 빠르게 조치를     │
│  완료해주세요.               │
│                             │
│                             │
├─────────────────────────────┤
│  [ 해당 사고 바로가기 ]      │  ← 관련 화면 이동 버튼
└─────────────────────────────┘
```

---

### 컴포넌트 상세

#### 헤더
- 좌: 뒤로가기 (←)
- 중: "알림 상세"

#### 알림 정보
- 유형 아이콘 + 유형명
- 수신 일시 (YYYY.MM.DD HH:MM)
- 구분선
- 알림 전문 (전체 표시, 말줄임 없음)

#### 하단 이동 버튼
- 알림 유형에 따라 이동 대상 다름:

| 알림 유형 | 이동 대상 |
|---|---|
| 미조치 기한 초과 | 해당 사고 상세 |
| 조치 촉구 | 해당 사고 상세 |
| 가입 승인 완료 | 간부 홈 |
| 가입 거절 | CMN-03 (거절 상태) |
| 공지사항 | CMN-07 (공지 상세) |
| 반려 | 해당 사고 수정 화면 |

- 이동 대상 없는 경우: 버튼 미표시

<a name="cmn-06"></a>
## CMN-06 · 공지사항 목록

**진입 경로:** 바텀 네비 > 공지사항 탭 (전 권한)

---

### 레이아웃

```
┌─────────────────────────────┐
│  공지사항              [작성]│  ← 헤더 ([작성]은 관리자만)
├─────────────────────────────┤
│                             │
│  ┌─────────────────────┐   │
│  │ 📌 [필독] 5월 안전   │   │  ← 공지 항목 (중요 공지)
│  │ 교육 일정 안내       │   │
│  │ 2026.05.15          │   │
│  └─────────────────────┘   │
│                             │
│  ┌─────────────────────┐   │
│  │ 아차사고 등록 방법   │   │  ← 일반 공지
│  │ 안내                │   │
│  │ 2026.05.10          │   │
│  └─────────────────────┘   │
│                             │
│  ┌─────────────────────┐   │
│  │ 4월 안전 교육 결과   │   │
│  │ 2026.04.30          │   │
│  └─────────────────────┘   │
│                             │
│  ... (스크롤)               │
│                             │
├─────────────────────────────┤
│  (권한별 바텀 네비)          │
└─────────────────────────────┘
```

---

### 컴포넌트 상세

#### 헤더
- 중: "공지사항"
- 우: [작성] 버튼 — **관리자만 표시**, 탭 시 공지 작성 화면으로 이동

#### 공지 항목
- 중요 공지: 📌 아이콘 + "[필독]" 접두사, 상단 고정 또는 강조 배경
- 일반 공지: 제목 + 등록 일자
- 제목: 2줄 말줄임
- 읽지 않은 공지: 제목 굵게 표시
- 항목 탭: CMN-07로 이동

#### 빈 상태
- "등록된 공지사항이 없어요"
