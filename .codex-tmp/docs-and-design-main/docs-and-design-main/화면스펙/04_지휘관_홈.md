---

### 상단바

- 좌: 앱 타이틀 "아차사고 관리"
- 우: 벨 아이콘 (🔔) — 읽지 않은 알림 시 숫자 뱃지
- 우: 햄버거 메뉴 (☰) — 탭 시 통계 그래프 화면으로 이동

### 바텀 네비게이션

- 홈 / 공지사항 / 내 정보 (3탭)
- 사고 등록·신고 현황 없음 (지휘관은 열람 전용)

## COM-02 · 고위험 부대 사고 현황 상세

**진입 경로:** COM-01 섹션 1 > [전체 보기] / 항목 탭

---

### 레이아웃

```
┌─────────────────────────────┐
│ [←]   리스크 평가           │
├─────────────────────────────┤
│  [부대별] [유형별]           │  ← 뷰 전환 탭
├─────────────────────────────┤
│                             │
│  집계 기준: 최근 30일        │  ← 집계 기간 안내
│                             │
│  ┌─────────────────────┐   │
│  │ 1위  ○○부대 1중대   │   │
│  │      ●●●●○ 위험도 높음│  │
│  │      사고 8건        │   │
│  │      감전 5 / 떨어짐 3│  │  ← 유형 분포
│  └─────────────────────┘   │
│  ┌─────────────────────┐   │
│  │ 2위  ○○부대 2중대   │   │
│  │      ●●●○○ 위험도 보통│  │
│  │      사고 5건        │   │
│  │      끼임 3 / 부딪힘 2│  │
│  └─────────────────────┘   │
│                             │
│  ... (전체 부대 목록)       │
│                             │
└─────────────────────────────┘
```

---

### 컴포넌트 상세

#### 헤더
- 좌: 뒤로가기 (←)
- 중: "리스크 평가"

#### 집계 기간 안내
- "집계 기준: 최근 30일" — 고정 텍스트 (관리자 설정 기간과 연동 가능하나 현재는 고정)

#### 부대별 항목
- 순위 + 부대명
- 위험도 점수 시각화 (5단계 도트) + 텍스트 수준 (낮음/보통/높음/매우 높음)
- 사고 건수
- 주요 사고 유형 분포 (건수 많은 순 2~3개)
- 항목 탭: 해당 부대 사고 목록 (ADM-02 유사 구조, 부대 필터 적용 상태)

#### 유형별 항목 (유형별 탭)
- 순위 + 사고 유형 아이콘 + 유형명
- 위험도 수준
- 전체 사고 중 건수 + 비율
- 항목 탭: 해당 유형 사고 목록

### 컴포넌트 상세

#### 헤더
- 좌: 뒤로가기 (←)
- 중: "[카테고리명] 사고 현황"

#### 요약 카드
- COM-01 섹션 2와 동일한 데이터 재표시 (건수 / 증감 / 비율)

#### 사고 목록
- 항목 구성: 유형 / 처리 상태 / 등록자 / 발생 장소 / 등록 일시
- 기한 초과 건: ⚠ 표시
- 정렬: 등록일 내림차순
- 항목 탭: 사고 상세 (읽기 전용 — 지휘관은 수정 불가)

#### 사고 상세 (지휘관 버전)
- MGT-02 구조와 동일
- **차이점**: 수정 버튼 / 조치 완료 처리 버튼 없음 (열람 전용)

│  기술자 점검 의뢰. 손상 배선 │

│                             │

### 컴포넌트 상세

#### 헤더
- 좌: 뒤로가기 (←)
- 중: "사례 상세"

#### 사진
- 등록 사진 있을 경우: 가로 스크롤 썸네일 표시
- 없을 경우: 사진 영역 미표시

#### 정보 항목
| 항목 | 내용 |
|---|---|
| 사고 유형 | 아이콘 + 유형명 |
| 사례 제목 | 큰 폰트, 멀티라인 |
| 발생 부대 | 부대명 |
| 발생 일시 | YYYY.MM.DD |
| 위험도 | 도트 + 수준 텍스트 |

#### 내용 섹션
- 식별된 위험 요인 (전체 표시, 말줄임 없음)
- 조치 결과 (전체 표시)
- AI 예방 대책 카드 (REG-06 카드와 동일 구조, 스크롤)

#### 하단 버튼
- 없음 (지휘관은 열람만 가능)

### 컴포넌트 상세

#### 헤더
- 좌: 뒤로가기 (←)
- 중: "의사결정 지원"

#### 긴급도 뱃지
| 수준 | 색상 |
|---|---|
| 즉시 권고 | 빨강 |
| 중기 권고 | 주황 |
| 장기 권고 | 파랑 |

#### 권고 배경
- 데이터 기반 상황 설명 (서버 로직 생성)
- 수치 강조 (증감률, 건수 등)

#### 권고 내용
- 넘버링 리스트 형태의 구체적 권고 항목

#### 근거 데이터
- 미니 차트: 관련 카테고리의 최근 6개월 추이 꺾은선 그래프
- [관련 사고 N건]: COM-03으로 이동 (해당 카테고리 사고 목록)

## 화면 간 이동 흐름 요약

```
COM-01 (지휘관 홈)
  ├─ 섹션 1 (리스크 평가)
  │   ├─ [전체 보기] ────────────────── COM-02 (고위험 부대 전체)
  │   └─ 항목 탭 ──────────────────── COM-02 (해당 부대 포커스)
  │
  ├─ 섹션 2 (중점 관리)
  │   └─ 카테고리 카드 탭 ────────── COM-03 (해당 카테고리 사고 목록)
  │
  ├─ 섹션 3 (타 부대 사례)
  │   ├─ 사례 카드 탭 ─────────────── COM-04 (사례 상세)
  │   └─ [전체 보기] ────────────────── 사례 전체 목록 → COM-04
  │
  └─ 섹션 4 (의사결정 지원)
      └─ 권고 카드 [상세] ──────────── COM-05 (권고 상세)

COM-02 (고위험 부대)
  └─ 항목 탭 ─────────────────────── 사고 목록 (열람 전용)

COM-03 (집중 카테고리 사고 목록)
  └─ 항목 탭 ─────────────────────── 사고 상세 (열람 전용, 수정·조치 버튼 없음)

COM-05 (의사결정 지원 상세)
  └─ [관련 사고 N건] ──────────────── COM-03

공통
  └─ 햄버거 메뉴 ─────────────────── 통계 그래프 화면
```
