<a name="reg-01"></a>
## REG-01 · 상황 설명 및 사진 등록

**진입 경로:** 홈 > 사고 등록 버튼 탭 / 바텀 네비 > 사고 등록

**다음 화면:** REG-02 (AI 추가 질문이 있을 때) / REG-05 (없을 때)

---

### 레이아웃

```
┌─────────────────────────────┐
│ [←]   사고 등록        1/5  │  ← 헤더
├─────────────────────────────┤
│  ●────○────○────○────○      │  ← 스텝 인디케이터
├─────────────────────────────┤
│                             │
│  [ 사진 첨부 영역 ]          │
│                             │
│  식별 일시                   │
│  [ 날짜 / 시간 피커 ]         │
│                             │
│  발생 장소                   │
│  [ 텍스트 필드 ]             │
│                             │
│  식별된 위험 요인 *           │
│  ┌─────────────────────┐    │
│  │                     │    │
│  │  (멀티라인 입력창)   │    │
│  │                     │    │
│  └─────────────────────┘    │
│                             │
├─────────────────────────────┤
│  [ 다음 ]                   │  ← 하단 고정
└─────────────────────────────┘
```

---

### 컴포넌트 상세

#### 헤더
- 좌: 뒤로가기 버튼 (←) — 탭 시 이전 화면으로 이동 (등록 중단 확인 다이얼로그)
- 중: 타이틀 "사고 등록"
- 우: 단계 표시 "1 / 5"

#### 스텝 인디케이터
- 도트 5개 (현재 단계 강조, 나머지 비활성)
- 도트 간 연결선

#### 사진 첨부 영역

**[사진 없는 상태]**
```
┌─────────────────────────────┐
│                             │
│        📷 카메라 아이콘      │
│    사진을 첨부해주세요 (선택) │
│                             │
│  [카메라]  [갤러리]  [건너뛰기] │
└─────────────────────────────┘
```
- 카메라 버튼: 카메라 앱 실행
- 갤러리 버튼: 갤러리 선택 화면
- 건너뛰기: 사진 없이 다음 필드로 포커스 이동

**[사진 첨부된 상태]**
```
┌──────┬──────┬──────┬──────┐
│ 썸네일│ 썸네일│ 썸네일│  +  │
│      │      │      │ 추가 │
└──────┴──────┴──────┴──────┘
     [재촬영]        [추가]
```
- 썸네일: 가로 스크롤 목록
- 썸네일 탭: 사진 크게 보기 (삭제 옵션 포함)
- 추가(+) 버튼: 카메라/갤러리 선택 시트
- 재촬영: 마지막 사진 재촬영

#### 식별 일시
- 라벨: "식별 일시"
- 입력: 날짜+시간 피커 (탭 시 OS 네이티브 피커 표시)
- 기본값: 현재 날짜·시각 자동 입력
- 형식: YYYY년 MM월 DD일 HH:MM

#### 발생 장소
- 라벨: "발생 장소"
- 입력: 단일 라인 텍스트 필드
- Placeholder: "예) 3중대 창고 앞"
- 필수 여부: 선택 (비워도 다음 진행 가능)

#### 식별된 위험 요인
- 라벨: "식별된 위험 요인 *" (필수 표시)
- 입력: 멀티라인 텍스트 입력창
- Placeholder: "어떤 위험 요인이 있었나요? 최대한 자세히 작성해주세요."
- 최소 높이: 120dp
- 최대 글자 수: 500자 (우측 하단 카운터 표시)

#### 하단 버튼
- "다음" — Primary, full-width
- **비활성 조건:** 위험 요인 텍스트가 비어있을 때
- **활성 조건:** 위험 요인 1자 이상 입력 시
- 탭 시: AI 분석 백그라운드 시작 + REG-02 or REG-05 진입

---

### 상태별 UI

| 상태 | UI 처리 |
|---|---|
| 초기 | 일시 자동 입력, 나머지 비어있음 |
| 키보드 노출 | 하단 버튼 키보드 위로 이동 |
| 사진 첨부 중 | 로딩 스피너 (썸네일 자리에 표시) |
| 사진 첨부 실패 | 토스트 "사진 첨부에 실패했어요" + 재시도 |

---

### 등록 중단 확인 다이얼로그 (뒤로가기 탭 시)
```
작성 중인 내용이 있어요.
등록을 중단할까요?

[계속 작성]  [중단하기]
```
- 계속 작성: 다이얼로그 닫기
- 중단하기: 홈으로 이동 (작성 내용 삭제)

<a name="reg-02"></a>
## REG-02 · AI 추가 질문

**진입 경로:** REG-01 > "다음" 탭 (AI 추가 질문이 존재할 때)

**다음 화면:** REG-02 (다음 질문) / REG-03 (마지막 질문 완료) / REG-04 (건너뛰기)

---

### 레이아웃

```
┌─────────────────────────────┐
│ [←]   사고 등록        2/5  │
├─────────────────────────────┤
│  ○────●────○────○────○      │
├─────────────────────────────┤
│  질문 2 / 4                 │  ← 질문 진행률
│  ████████░░░░░░░░           │  ← 진행률 바
│                             │
│  이 사고가 발생한            │
│  시간대는 언제인가요?         │  ← 질문 텍스트
│                             │
│  ┌──────────────────────┐   │
│  │  오전 (06:00~12:00)  │   │  ← 선택지 버튼
│  └──────────────────────┘   │
│  ┌──────────────────────┐   │
│  │  오후 (12:00~18:00)  │   │
│  └──────────────────────┘   │
│  ┌──────────────────────┐   │
│  │  야간 (18:00~06:00)  │   │
│  └──────────────────────┘   │
│                             │
│  ─────── 또는 ───────       │
│                             │
│  ┌──────────────────────┐   │
│  │  직접 입력...         │   │  ← 직접 입력창
│  └──────────────────────┘   │
│                             │
│  나머지 질문 건너뛰기         │  ← 텍스트 버튼
├─────────────────────────────┤
│  [ 다음 질문 ]              │
└─────────────────────────────┘
```

---

### 컴포넌트 상세

#### 질문 진행률
- 텍스트: "질문 N / 전체"
- 진행률 바: 현재 질문 기준 채워짐

#### 질문 텍스트
- 폰트: 제목 크기 (18~20sp)
- 멀티라인 지원

#### 선택지 버튼
- full-width, 탭 시 선택 상태 전환
- 선택된 항목: 강조 색상 + 체크 아이콘
- 단일 선택 (라디오 동작)
- 선택지 탭 후 직접 입력창에 내용 있으면 입력창 초기화

#### 직접 입력창
- Placeholder: "직접 입력해주세요 (선택)"
- 단일 라인
- 내용 입력 시 선택지 버튼 선택 해제

#### "나머지 질문 건너뛰기" 버튼
- 텍스트 버튼 (작은 폰트, secondary 색상)
- 탭 시: REG-04로 이동 (로딩 없이)

#### "다음 질문" 버튼
- Primary, full-width
- **비활성 조건:** 선택지 미선택 + 직접 입력 비어있음
- **활성 조건:** 선택지 선택됨 OR 직접 입력 내용 있음
- 마지막 질문에서 탭 시: REG-03으로 이동

---

### 상태별 UI

| 상태 | UI 처리 |
|---|---|
| 선택지 선택 후 | 선택 항목 강조, 직접 입력창 비활성(회색) |
| 직접 입력 중 | 선택지 버튼 전체 비선택 상태로 초기화 |
| 마지막 질문 | "다음 질문" → "분석 시작" 문구로 변경 |

<a name="reg-04"></a>
## REG-04 · 사고 유형 선택

**진입 경로:** REG-03 (분석 완료) / REG-02 (건너뛰기)

**다음 화면:** REG-05

---

### 레이아웃

```
┌─────────────────────────────┐
│ [←]   사고 등록        2/5  │
├─────────────────────────────┤
│  ○────●────○────○────○      │
├─────────────────────────────┤
│                             │
│  이 사고의 유형을            │
│  선택해주세요                │
│                             │
│  ┌───────────────────────┐  │
│  │ 💡 AI가 [감전]을 추천  │  │  ← AI 추천 배너 (추천 시에만)
│  │    했어요              │  │
│  └───────────────────────┘  │
│                             │
│  ┌──────────┐ ┌──────────┐  │
│  │  🔥 화재 │ │  💥 폭발 │  │
│  └──────────┘ └──────────┘  │
│  ┌──────────┐ ┌──────────┐  │
│  │ ⬇ 떨어짐 │ │  ⚙ 끼임  │  │
│  └──────────┘ └──────────┘  │
│  ┌──────────┐ ┌──────────┐  │
│  │ 🤜 부딪힘│ │  🧱 붕괴 │  │
│  └──────────┘ └──────────┘  │
│  ┌──────────┐ ┌──────────┐  │
│  │⚡감전    │ │  ☁ 중독  │  │  ← AI 추천 항목: [AI 추천] 배지
│  │[AI 추천] │ │          │  │
│  └──────────┘ └──────────┘  │
│  ┌──────────┐ ┌──────────┐  │
│  │ 🌡 온열  │ │  ··· 기타│  │
│  └──────────┘ └──────────┘  │
│                             │
├─────────────────────────────┤
│  [ 다음 ]                   │
└─────────────────────────────┘
```

---

### 컴포넌트 상세

#### AI 추천 배너
- 표시 조건: AI 분석 성공 시에만 표시
- 미표시 조건: 건너뛰기로 진입 / 분석 실패 / 타임아웃
- 내용: "AI가 [유형명]을 추천했어요"
- 스타일: 강조 색상 배경 카드

#### 카테고리 그리드
- 2열 그리드
- 각 항목: 아이콘 + 텍스트 라벨
- AI 추천 항목: 우측 상단에 [AI 추천] 칩 배지
- 선택된 항목: 강조 테두리 + 배경색 변경
- 단일 선택 (라디오 동작)
- AI 추천 항목은 기본 선택 상태로 진입

#### 카테고리 목록
1. 화재
2. 폭발
3. 떨어짐 (추락)
4. 끼임 (협착)
5. 부딪힘 (충격)
6. 붕괴
7. 감전
8. 중독 (질식)
9. 온열질환
10. 기타

#### 하단 버튼
- "다음" — Primary, full-width
- **비활성 조건:** 아무것도 선택 안 됨
- **활성 조건:** 항목 선택됨
- AI 추천 항목 기본 선택이므로 진입 즉시 활성

---

### 상태별 UI

| 상태 | UI 처리 |
|---|---|
| AI 추천 있음 | 배너 표시 + 해당 항목 기본 선택 + [AI 추천] 배지 |
| AI 추천 없음 | 배너 미표시, 배지 없음, 기본 선택 없음 |

- 조치 완료 상태: 선택한 카드 내용 or 직접 입력 텍스트 표시

```

### 건너뛰기 / 실패 단락 경로

| 상황 | 경로 |
|---|---|

| 3단계 예방대책 건너뛰기 | REG-05 실패 → REG-07 (카드 없음, 직접 입력 모드 자동 전환) |

│  │ 미끄러짐             │   │
│  │ 조치: 안전화 착용    │   │
│  │        미끄럼 방지대│   │
│  └─────────────────────┘   │
│                             │
│  ... (스크롤)               │
│                             │
└─────────────────────────────┘
```

### 컴포넌트 상세

| 컴포넌트     | 내용                                                                               |
|--------------|------------------------------------------------------------------------------------|
| 검색 바      | 발생 상황·조치 결과 키워드 검색, 실시간 필터링                                     |
| 유형 필터 칩 | 전체 + 10개 유형, 가로 스크롤, 단일 선택                                           |
| 사례 카드    | 사고 유형 아이콘 + 유형명 / 위험도 배지 / 발생 상황 요약(2줄) / 조치 결과 요약(2줄)|
| 조회 범위    | 조치 완료 상태(공개)인 사고만 표시, 작성자 개인정보(이름·군번) 비표시             |
| 빈 상태      | "조건에 맞는 사례가 없어요" 안내                                                   |

### 상태

| 상태        | 표시                              |
|-------------|-----------------------------------|
| 초기 진입   | 현재 사고 유형 자동 선택 (AI 추천 유형 기반) |
| 검색 결과 없음 | "검색 결과가 없어요" 안내       |
| 로딩 중     | 스켈레톤 카드 3개                 |

### 인터랙션

- 진입 시: REG-04에서 선택된 사고 유형 필터 자동 적용
- 필터 칩 전환 시 목록 즉시 갱신
- 카드 탭 → REG-11
- 뒤로가기 → REG-07 복귀 (모드·입력 내용 유지)

## REG-11 · 유사 사례 상세

**진입 경로:** REG-10 > 사례 카드 탭

**다음 화면:** REG-10 (뒤로가기)

---

### 레이아웃

```
┌─────────────────────────────┐
│ [←]   사례 상세              │  ← 상단 바
├─────────────────────────────┤
│                             │
│  ⚡ 감전            [높음]   │  ← 유형 + 위험도 배지
│                             │
│  ─── 발생 상황 ─────────── │
│                             │
│  배전반 점검 작업 중 절연이  │
│  손상된 전선에 맨손으로      │
│  접촉하여 감전 위기 발생     │
│                             │
│  ─── AI 예방 대책 ────────  │
│                             │
│  ┌─────────────────────┐   │
│  │  [ AI 이미지 ]       │   │  ← 예방 대책 카드 1
│  ├─────────────────────┤   │
│  │ 절연 장갑 착용 필수   │   │
│  └─────────────────────┘   │
│                             │
│  ┌─────────────────────┐   │
│  │  [ AI 이미지 ]       │   │  ← 예방 대책 카드 2
│  ├─────────────────────┤   │
│  │ 작업 전 절연 상태    │   │
│  │ 점검 필수            │   │
│  └─────────────────────┘   │
│                             │
│  ─── 조치 결과 ─────────── │
│                             │
│  안전장갑 착용 및 전선 절연  │
│  테이프 즉시 처리, 안전 표지 │
│  판 설치                     │
│                             │
│  ─── 관련 사진 ─────────── │
│                             │
│  ┌─────┐ ┌─────┐           │
│  │사진1│ │사진2│           │  ← 가로 스크롤 썸네일
│  └─────┘ └─────┘           │
│                             │
└─────────────────────────────┘
```

### 컴포넌트 상세

| 컴포넌트       | 내용                                                             |
|----------------|------------------------------------------------------------------|
| 유형 + 위험도  | 사고 유형 아이콘 + 유형명, 위험도 배지 (낮음/보통/높음/매우 높음)|
| 발생 상황      | 전체 텍스트 표시 (말줄임 없음)                                   |
| AI 예방 대책   | AI 생성 이미지 + 예방 대책 텍스트 카드 (스크롤)                  |
| 조치 결과      | 전체 텍스트 표시                                                 |
| 관련 사진      | 등록 사진 + 조치 완료 사진 썸네일 (탭 시 풀스크린)               |
| 수정 버튼      | 없음 — 열람 전용                                                 |

### 인터랙션

- 뒤로가기 → REG-10 복귀
- 사진 썸네일 탭 → 풀스크린 이미지 뷰어 (좌우 스와이프)
