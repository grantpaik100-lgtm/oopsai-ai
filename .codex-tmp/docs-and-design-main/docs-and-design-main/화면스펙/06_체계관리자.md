# 화면 스펙 — 체계관리자

> 바텀 네비: **홈 / 공지사항 / 내 정보** (3탭)
> 공지사항(CMN-06~07) · 내 정보(CMN-08~10)는 공통 화면 스펙 참고
> 알림: 전 권한 공통 상단 벨 아이콘 (CMN-04~05 참고)

---

## 화면 목록

| ID     | 화면명              | 진입 경로                        |
|--------|---------------------|----------------------------------|
| [SYS-01](#sys-01) | 체계관리자 홈       | 로그인 성공 → 체계관리자 권한    |
| [SYS-02](#sys-02) | 관리자 계정 목록    | SYS-01 계정 관리 버튼            |
| [SYS-03](#sys-03) | 계정 상세           | SYS-02 계정 항목 탭              |
| [SYS-04](#sys-04) | 계정 생성           | SYS-02 [+] 버튼                  |

<a name="sys-01"></a>
## SYS-01 체계관리자 홈

### 레이아웃

```
┌─────────────────────────────────┐
│ 🔔          아차사고 관리       │  ← 상단 바 (벨 아이콘)
├─────────────────────────────────┤
│                                 │
│  안녕하세요, [이름] [계급]님    │  ← 환영 메시지
│                                 │
│ ┌─────────────────────────────┐ │
│ │  📊 관리자 계정 현황         │ │  ← 현황 카드
│ │                             │ │
│ │  전체   인사담당자  안전담당자│ │
│ │   12       7          5     │ │
│ │                             │ │
│ │       [계정 관리 →]         │ │  ← 버튼
│ └─────────────────────────────┘ │
│                                 │
│ ┌─────────────────────────────┐ │
│ │  ⚙️  시스템 정보             │ │  ← 시스템 정보 카드
│ │                             │ │
│ │  앱 버전     1.0.0          │ │
│ │  서버 상태   정상 ✅         │ │
│ └─────────────────────────────┘ │
│                                 │
├─────────────────────────────────┤
│  🏠홈  │  📢공지  │  👤내정보  │  ← 바텀 네비 (3탭)
└─────────────────────────────────┘
```

### 컴포넌트 상세

| 컴포넌트        | 내용                                                                 |
|-----------------|----------------------------------------------------------------------|
| 환영 메시지     | "[이름] [계급]님" 형식, 로그인 정보 기반                             |
| 관리자 현황 카드| 전체 / 인사담당자 / 안전담당자 계정 수 표시 (서버 실시간 조회)       |
| 계정 관리 버튼  | 탭 시 SYS-02 이동                                                    |
| 시스템 정보 카드| 앱 버전(하드코딩), 서버 상태(API health check 결과: 정상/점검중)     |
| 벨 아이콘       | 미읽음 알림 수 뱃지 표시 → CMN-04 이동                              |

### 상태

| 상태          | 표시                                    |
|---------------|-----------------------------------------|
| 서버 정상     | "정상 ✅" 녹색 텍스트                   |
| 서버 점검중   | "점검중 ⚠️" 주황색 텍스트               |
| 데이터 로딩중 | 현황 카드 숫자 자리 스켈레톤 표시       |
| 조회 실패     | "–" 표시 + 새로고침 버튼               |

### 인터랙션

- 화면 진입 시 계정 현황 API 호출
- 풀-투-리프레시(Pull to refresh) 지원
- [계정 관리 →] 탭 → SYS-02

<a name="sys-02"></a>
## SYS-02 관리자 계정 목록

### 레이아웃

```
┌─────────────────────────────────┐
│ ← 계정 관리                 [+] │  ← 상단 바 (뒤로 / 생성 버튼)
├─────────────────────────────────┤
│ [전체 12] [인사담당자 7] [안전담당자 5] │  ← 필터 탭
├─────────────────────────────────┤
│ 🔍 이름·군번 검색               │  ← 검색 바
├─────────────────────────────────┤
│ ┌─────────────────────────────┐ │
│ │ 홍길동 중사                 │ │  ← 계정 항목
│ │ 군번: 12345678              │ │
│ │ 소속: 00대대                │ │
│ │ 권한: 인사담당자     [>]    │ │
│ └─────────────────────────────┘ │
│ ┌─────────────────────────────┐ │
│ │ 김철수 원사                 │ │
│ │ 군번: 87654321              │ │
│ │ 소속: 01대대                │ │
│ │ 권한: 안전담당자     [>]    │ │
│ └─────────────────────────────┘ │
│         (스크롤)                │
└─────────────────────────────────┘
```

### 컴포넌트 상세

| 컴포넌트       | 내용                                                              |
|----------------|-------------------------------------------------------------------|
| 필터 탭        | 전체 / 인사담당자 / 안전담당자 — 각 탭 이름 우측에 계정 수 표시  |
| 검색 바        | 이름 또는 군번으로 실시간 필터링                                  |
| 계정 항목      | 이름 + 계급 / 군번 / 소속 / 권한 표시, 탭 시 SYS-03 이동         |
| [+] 버튼       | 상단 우측, 탭 시 SYS-04 이동                                     |

### 상태

| 상태        | 표시                                     |
|-------------|------------------------------------------|
| 목록 없음   | "등록된 관리자 계정이 없습니다" 안내     |
| 검색 결과 없음 | "검색 결과가 없습니다" 안내          |
| 로딩중      | 스켈레톤 카드 3개                        |

### 인터랙션

- 필터 탭 전환 시 목록 즉시 갱신
- 검색어 입력 시 실시간 필터 (로컬)
- 항목 탭 → SYS-03 이동
- [+] 탭 → SYS-04 이동

<a name="sys-03"></a>
## SYS-03 계정 상세

### 레이아웃

```
┌─────────────────────────────────┐
│ ← 계정 상세                     │  ← 상단 바
├─────────────────────────────────┤
│                                 │
│  👤 홍길동 중사                  │  ← 프로필 헤더
│     인사담당자                  │  ← 권한 배지
│                                 │
│ ┌─────────────────────────────┐ │
│ │ 이름     홍길동              │ │  ← 정보 (수정 불가, 회색)
│ │ 계급     중사                │ │
│ │ 군번     12345678            │ │
│ │ 소속     00대대              │ │
│ │ 가입일   2025.01.15          │ │
│ └─────────────────────────────┘ │
│                                 │
│ ┌─────────────────────────────┐ │
│ │ 권한     인사담당자     [변경]│ │  ← 권한 영역
│ └─────────────────────────────┘ │
│                                 │
│ [권한 회수]          [계정 삭제] │  ← 액션 버튼 (하단)
│                                 │
└─────────────────────────────────┘
```

### 컴포넌트 상세

| 컴포넌트       | 내용                                                                           |
|----------------|--------------------------------------------------------------------------------|
| 프로필 헤더    | 이름 + 계급, 권한 배지 (인사담당자 / 안전담당자)                               |
| 정보 필드      | 이름·계급·군번·소속·가입일 — 모두 읽기 전용 (회색 배경)                        |
| 권한 [변경]    | 탭 시 권한 변경 바텀시트 표시                                                   |
| [권한 회수]    | 탭 시 확인 다이얼로그 → 확인 시 계정 비활성화 처리                              |
| [계정 삭제]    | 탭 시 확인 다이얼로그 → 확인 시 계정 영구 삭제, SYS-02로 복귀                   |

### 권한 변경 바텀시트

```
┌─────────────────────────────────┐
│        권한 변경                │
│─────────────────────────────────│
│  ○ 인사담당자  (현재 선택)      │
│  ○ 안전담당자                   │
│                                 │
│         [저장]                  │
└─────────────────────────────────┘
```

- 현재 권한 기본 선택 상태
- [저장] 탭 시 권한 변경 API 호출 → 완료 토스트 → 바텀시트 닫힘 → 화면 권한 배지 갱신

### 권한 회수 다이얼로그

```
┌────────────────────────┐
│  권한을 회수하시겠습니까?│
│                        │
│  홍길동 중사의 관리자   │
│  권한이 회수됩니다.     │
│                        │
│  [취소]      [회수]    │
└────────────────────────┘
```

- [회수] 탭 → 계정 상태 "비활성" 처리 → 완료 토스트 → SYS-02로 복귀

### 계정 삭제 다이얼로그

```
┌────────────────────────┐
│  계정을 삭제하시겠습니까?│
│                        │
│  삭제 후 복구가         │
│  불가능합니다.          │
│                        │
│  [취소]      [삭제]    │
└────────────────────────┘
```

- [삭제] 탭 → 계정 영구 삭제 → 완료 토스트 → SYS-02로 복귀

### 상태

| 상태          | 표시                                        |
|---------------|---------------------------------------------|
| 정상 계정     | 기본 표시                                   |
| 비활성 계정   | 상단에 "비활성 계정" 배너, [권한 회수] 비활성|
| 권한 변경 중  | [저장] 버튼 로딩 스피너, 비활성             |
| 삭제 처리 중  | [계정 삭제] 버튼 로딩 스피너                |

<a name="sys-04"></a>
## SYS-04 계정 생성

### 레이아웃

```
┌─────────────────────────────────┐
│ ← 계정 생성                     │  ← 상단 바
├─────────────────────────────────┤
│                                 │
│  계급 *                         │
│  ┌─────────────────────────┐    │
│  │ 계급 선택 (드롭다운) ▼  │    │
│  └─────────────────────────┘    │
│                                 │
│  이름 *                         │
│  ┌─────────────────────────┐    │
│  │                         │    │
│  └─────────────────────────┘    │
│                                 │
│  군번 *                         │
│  ┌─────────────────────────┐    │
│  │ 숫자 8자리              │    │
│  └─────────────────────────┘    │
│                                 │
│  소속 부대 *                    │
│  ┌─────────────────────────┐    │
│  │                         │    │
│  └─────────────────────────┘    │
│                                 │
│  아이디 *                       │
│  ┌─────────────────────────┐    │
│  │                         │    │
│  └─────────────────────────┘    │
│                                 │
│  임시 비밀번호 *                │
│  ┌─────────────────────────┐    │
│  │ 자동 생성됨 (수정 가능) │    │
│  └─────────────────────────┘    │
│                                 │
│  권한 *                         │
│  ┌─────────────────────────┐    │
│  │ ○ 인사담당자            │    │
│  │ ○ 안전담당자            │    │
│  └─────────────────────────┘    │
│                                 │
│         [생성]                  │  ← 하단 CTA
│                                 │
└─────────────────────────────────┘
```

### 컴포넌트 상세

| 컴포넌트       | 내용                                                                                 |
|----------------|--------------------------------------------------------------------------------------|
| 계급           | 드롭다운 선택 (이병~대령 등 군 계급 목록)                                            |
| 이름           | 텍스트 입력, 최대 10자                                                               |
| 군번           | 숫자 전용, 8자리 고정 (미달/초과 시 오류)                                            |
| 소속 부대      | 텍스트 입력 (부대명 직접 입력)                                                       |
| 아이디         | 영숫자 조합, 중복 확인은 생성 시 서버에서 처리                                       |
| 임시 비밀번호  | 자동 생성된 값 표시 (수정 가능), 생성 완료 후 계정 상세에서 확인 가능               |
| 권한           | 라디오 버튼 — 인사담당자 / 안전담당자 (둘 중 하나 필수 선택)                        |
| [생성] 버튼    | 필수 필드 모두 입력 시 활성화, 탭 시 계정 생성 API 호출                              |

### 유효성 검사

| 필드        | 규칙                                          | 오류 메시지                      |
|-------------|-----------------------------------------------|----------------------------------|
| 계급        | 필수 선택                                     | "계급을 선택해주세요"            |
| 이름        | 필수, 1~10자                                  | "이름을 입력해주세요"            |
| 군번        | 필수, 숫자 8자리                              | "군번은 숫자 8자리입니다"        |
| 소속 부대   | 필수, 1~30자                                  | "소속 부대를 입력해주세요"       |
| 아이디      | 필수, 영숫자 4~20자                           | "아이디를 입력해주세요"          |
| 임시 비밀번호| 영문+숫자 8자 이상                           | "비밀번호는 8자 이상입니다"      |
| 권한        | 필수 선택                                     | "권한을 선택해주세요"            |
| 중복 아이디 | 서버 응답 오류                                | "이미 사용 중인 아이디입니다"    |

### 상태

| 상태            | 표시                                                    |
|-----------------|---------------------------------------------------------|
| 초기            | 임시 비밀번호만 자동 값 표시, 나머지 빈 필드            |
| 필수 미입력     | [생성] 버튼 비활성                                      |
| 필수 모두 입력  | [생성] 버튼 활성                                        |
| 생성 중         | [생성] 버튼 로딩 스피너 + 비활성                        |
| 생성 성공       | "계정이 생성되었습니다" 토스트 → SYS-02 복귀           |
| 생성 실패       | 오류 토스트 표시, 화면 유지                             |

### 인터랙션

- 임시 비밀번호: 화면 진입 시 자동 생성 (영문 대소문자+숫자 조합 12자), 복사 아이콘 탭으로 클립보드 복사 가능
- [생성] 버튼: 모든 필수 필드 입력 완료 시 활성화
- 생성 완료 후 SYS-02로 자동 복귀, 목록 상단에 새 계정 표시

## 화면 간 이동 흐름

```
로그인 성공 (체계관리자)
└── SYS-01 체계관리자 홈
    └── [계정 관리 →]
        └── SYS-02 관리자 계정 목록
            ├── [+] 생성 버튼
            │   └── SYS-04 계정 생성
            │       └── [생성] 성공 → SYS-02 복귀
            └── 계정 항목 탭
                └── SYS-03 계정 상세
                    ├── [변경] 탭 → 권한 변경 바텀시트 → 저장 → SYS-03 갱신
                    ├── [권한 회수] → 다이얼로그 → 확인 → SYS-02 복귀
                    └── [계정 삭제] → 다이얼로그 → 확인 → SYS-02 복귀
```
