# 클라우드 인프라 견적 — 대회 시연용 최소 비용

> 기준일: 2026-05-16 · 환율: 1 USD ≈ ₩1,350
> 목적: **소수 인원 시연** (동시 사용자 5~10명, 총 등록 30~50건 가정)
> 전략: 네트워크 속도 제외, **비용 최소 + 가용 성능 최대화**

---

## 1. 시스템 구성 전제

| 컴포넌트 | 역할 | 비고 |
|---------|------|------|
| 앱 서버 | FastAPI (Python) · REST API | 최소 2 GB RAM 필요 |
| DB | PostgreSQL + pgvector | 앱 서버와 동일 VM에 자체 설치 → 관리형 DB 비용 제거 |
| 오브젝트 스토리지 | 사진 파일 저장 (Presigned URL) | 시연용 ≈ 5 GB |
| AI API | OpenAI (GPT-4o Vision · DALL·E 3 · Embedding) | 사용량 종량제 |
| FCM | 푸시 알림 | **무료** |

> **핵심 절감 포인트:** 관리형 DB(RDS/Cloud SQL)를 쓰지 않고 **VM에 PostgreSQL 직접 설치**.
> 대회 시연 규모에서 단일 VM으로 충분하며, DB 비용을 ₩15,000~25,000/월 절감.

## 2. 클라우드 인프라 견적 비교

### 2-A. GCP (Google Cloud Platform) — **최저가 추천**

| 항목 | 스펙 | 월 요금 (USD) | 월 요금 (₩) |
|------|------|-------------|------------|
| Compute Engine **e2-small** | 2 vCPU(공유) · 2 GB RAM | $15.69 | **₩21,182** |
| 영구 디스크 (SSD 20 GB) | OS + DB 데이터 | ~$3.40 | ~₩4,590 |
| Cloud Storage (5 GB) | 사진 파일 · $0.020/GB | ~$0.10 | ~₩135 |
| 네트워크 송신 (10 GB) | 시연 기간 예상 트래픽 | ~$1.00 | ~₩1,350 |
| **합계** | | **~$20.19** | **~₩27,257** |

> **e2-small 스펙:** 2 vCPU 공유 · 2 GB RAM · 최대 2 Gbps · 서울(asia-northeast3)
> **일 환산:** 약 ₩908/일 → 3일 시연 시 약 ₩2,724

#### 주의사항
- e2-small은 **공유 vCPU** — AI 요청 처리 시 CPU 스파이크 발생 가능
- RAM 2 GB: FastAPI + PostgreSQL 동시 운영 가능하나, `shared_buffers` 등 PostgreSQL 튜닝 필요
- 더 안정적으로 하려면 **e2-medium (4 GB, $31.38/월 ≈ ₩42,363)** 권장

### 2-B. AWS (Amazon Web Services)

| 항목 | 스펙 | 월 요금 (USD) | 월 요금 (₩) |
|------|------|-------------|------------|
| EC2 **t3.small** | 2 vCPU · 2 GB RAM | $18.98 | **₩25,623** |
| EBS gp3 (20 GB) | OS + DB 데이터 | ~$1.60 | ~₩2,160 |
| S3 Standard (5 GB) | 사진 파일 · $0.025/GB | ~$0.13 | ~₩176 |
| 네트워크 송신 (10 GB) | 시연 기간 예상 트래픽 | ~$0.90 | ~₩1,215 |
| **합계** | | **~$21.61** | **~₩29,174** |

> **t3.small 스펙:** 2 vCPU(버스트) · 2 GB RAM · 서울(ap-northeast-2)
> **일 환산:** 약 ₩972/일 → 3일 시연 시 약 ₩2,916

#### 주의사항
- t3 계열은 **CPU 크레딧 버스트** 방식 — 지속적 고부하 시 크레딧 소진 후 성능 저하
- **신규 계정 AWS Free Tier 대상자라면:**
  - t2.micro (1 vCPU · 1 GB) 750시간/월 **무료** (단, RAM 1 GB는 PostgreSQL+FastAPI에 빠듯)
  - EBS 30 GB 무료
  - → 실질 인프라 비용 **₩0** 가능 (Free Tier 12개월 이내 한정)

## 4. OpenAI API 비용 추정

> 등록 1건 기준 AI 호출 흐름: 이미지 분석 → 유형 분류 → 예방 대책(DALL·E 3 × 3장) → 임베딩

| 호출 | 모델 | 예상 토큰/비용 | 건당 비용 |
|------|------|--------------|---------|
| 이미지 분석 (POST /ai/analyze) | GPT-4o Vision | 이미지 765토큰 + 텍스트 500토큰 input / 300 output | ~$0.008 |
| 유형 분류 (POST /ai/classify) | GPT-4o | 600 input / 150 output | ~$0.003 |
| 예방 대책 생성 (POST /ai/prevention) | GPT-4o | 800 input / 600 output | ~$0.008 |
| 예방 대책 이미지 × 3장 | DALL·E 3 (1024×1024) | $0.040/장 | ~$0.120 |
| 임베딩 (POST /accidents) | text-embedding-3-small | ~150 tokens | ~$0.000003 |
| **건당 합계** | | | **≈ $0.14~0.20** |

| 시나리오 | 등록 건수 | 예상 AI 비용 |
|---------|---------|------------|
| 최소 (심사 시연) | 10건 | ~$1.50 ≈ ₩2,025 |
| 일반 (대회 당일 테스트 포함) | 30건 | ~$5.00 ≈ ₩6,750 |

> DALL·E 3 요금: $0.04/image (1024×1024 Standard)
> text-embedding-3-small: $0.02/1M tokens

## 5. 최종 추천 견적 (1개월 기준)

### 추천: GCP e2-medium (서울) — 안정성·비용 최적 균형

| 항목 | 금액 |
|------|------|
| GCP e2-medium (2 vCPU 공유 · 4 GB) | ₩42,363 |
| 영구 디스크 SSD 20 GB | ₩4,590 |
| Cloud Storage 5 GB | ₩135 |
| 네트워크 송신 10 GB | ₩1,350 |
| **인프라 소계** | **₩48,438** |
| OpenAI API (30건 등록 기준) | ₩6,750 |
| **총합계 (1개월)** | **≈ ₩55,188** |

> e2-medium을 추천하는 이유: e2-small(2 GB)은 FastAPI + PostgreSQL 동시 운영 시 메모리 부족 위험.
> 4 GB는 여유 있게 두 프로세스를 수용하고, 이미지 처리 시 메모리 급증도 흡수 가능.

### 최저가 대안: GCP e2-small (2 GB) — 비용 우선

| 항목 | 금액 |
|------|------|
| GCP e2-small (2 vCPU 공유 · 2 GB) | ₩21,182 |
| 영구 디스크 SSD 20 GB | ₩4,590 |
| Cloud Storage 5 GB | ₩135 |
| 네트워크 송신 10 GB | ₩1,350 |
| OpenAI API (30건 등록 기준) | ₩6,750 |
| **총합계 (1개월)** | **≈ ₩34,007** |

> PostgreSQL `shared_buffers=256MB`, `work_mem=4MB`로 튜닝 필수.
> uvicorn 워커 수 1~2개로 제한하여 메모리 관리.

## 6. 비용 절감 추가 팁

| 방법 | 절감 효과 |
|------|---------|
| **DALL·E 3 → GPT-4o 텍스트만 생성** | 이미지 생성 제거 시 건당 $0.12 절감 → 30건 기준 ₩4,860 절감 |
| **GCP Spot VM** (선점형) | e2-medium 기준 60~80% 할인, 단 중단 위험 있음 (대회 당일 비추천) |
| **GCP 약정 사용 할인 (1년 CUD)** | e2-medium ₩42,363 → ₩26,688 (37% 할인), 장기 운영 시 유리 |
| **PostgreSQL on VM vs. Cloud SQL** | Cloud SQL f1-micro $26/월 대비 자체 설치로 ₩35,100/월 절감 |
| **이미지 Presigned URL 직접 업로드** | 서버 트래픽·처리 부하 제거 (이미 설계에 반영됨) |
