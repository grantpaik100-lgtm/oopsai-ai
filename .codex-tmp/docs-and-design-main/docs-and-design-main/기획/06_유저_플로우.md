# User Flow (전체)

> 핵심 기능 플로우는 [05_MVP기능플로우.md](05_MVP기능플로우.md) 참고

---

## 계정 / 인증

### 1. 회원가입

```
로그인 화면
└── 회원가입 버튼 탭
    └── 회원가입 화면
        ├── 계급 / 이름 / 군번 입력
        ├── 소속 부대 입력
        ├── 아이디 / 비밀번호 입력
        └── 가입 신청 버튼
            ├── [성공]
            │   └── "관리자 승인 대기 중입니다" 안내 화면
            └── [실패 - 중복 아이디]
                └── 오류 안내 토스트
                    └── 회원가입 화면 유지
```

---

### 2. 로그인

```
앱 실행
└── 로그인 화면
    └── 아이디 / 비밀번호 입력
        └── 로그인 버튼 탭
            ├── [성공] → 권한 분기 플로우로 이동
            ├── [승인 대기 중]
            │   └── "관리자 승인 대기 중입니다" 안내 화면
            └── [실패]
                └── 오류 안내 토스트
                    └── 로그인 화면 유지
```

---

### 3. 로그아웃

```
내 정보
└── 로그아웃 버튼 탭
    └── 로그아웃 확인 다이얼로그
        ├── [확인]
        │   └── 로그인 화면으로 이동
        └── [취소]
            └── 내 정보 화면 유지
```

---

### 4. 비밀번호 변경

```
내 정보
└── 비밀번호 변경 탭
    └── 비밀번호 변경 화면
        ├── 현재 비밀번호 입력
        ├── 새 비밀번호 입력
        └── 새 비밀번호 확인 입력
            ├── [성공]
            │   └── 변경 완료 안내 토스트
            │       └── 내 정보 화면으로 이동
            ├── [현재 비밀번호 불일치]
            │   └── 오류 안내 토스트
            │       └── 비밀번호 변경 화면 유지
            └── [새 비밀번호 불일치]
                └── 오류 안내 토스트
                    └── 비밀번호 변경 화면 유지
```

---

### 5. 회원 탈퇴

```
내 정보
└── 회원 탈퇴 버튼 탭
    └── 탈퇴 확인 다이얼로그
        ├── [확인]
        │   └── 비밀번호 재확인 입력
        │       ├── [성공]
        │       │   └── 탈퇴 완료 처리
        │       │       └── 로그인 화면으로 이동
        │       └── [실패]
        │           └── 오류 안내 토스트
        │               └── 비밀번호 재확인 화면 유지
        └── [취소]
            └── 내 정보 화면 유지
```

---

### 6. 권한 분기 플로우

```
앱 실행
└── 로그인 화면
    └── 로그인 시도
        ├── [성공]
        │   └── 서버: 권한 확인
        │       ├── 간부 → 간부 홈
        │       ├── 관리자 → 관리자 홈
        │       ├── 지휘관 → 지휘관 홈
        │       └── 체계관리자 → 체계관리자 홈
        ├── [승인 대기 중]
        │   └── "관리자 승인 대기 중입니다" 안내 화면
        └── [로그인 실패]
            └── 오류 안내 토스트
                └── 로그인 화면 유지
```

---

### 7. 관리자 승인 대기 플로우

```
회원가입 완료
└── "승인 대기 중" 안내 화면 표시
    └── 서버: 관리자에게 신규 가입 신청 푸시 알림 발송
        └── 관리자: 간부 계정 승인 플로우 진행
            ├── [승인]
            │   └── 간부에게 승인 완료 푸시 알림 발송
            │       └── 간부: 앱 재진입 시 간부 홈으로 이동
            └── [거절]
                └── 승인 거절 플로우로 이동
```

---

### 8. 승인 거절 플로우

```
관리자: 간부 가입 승인 목록
└── 승인 상세 화면
    └── 거절 버튼 탭
        └── 거절 사유 입력 (선택)
            └── 거절 확인 버튼
                └── 서버: 계정 거절 처리
                    └── 간부에게 거절 푸시 알림 발송
                        └── 간부: 알림 확인
                            └── "가입이 거절되었습니다" 안내
                                └── 로그인 화면으로 이동
```

## 사고 조회 / 관리

### 9. 사고 상세 조회

```
내 사고 관리 / 전체 신고 현황
└── 사고 목록에서 항목 탭
    └── 사고 상세 화면
        ├── 등록 사진
        ├── AI 생성 사진
        ├── 조치 완료 사진 (조치 완료 시)
        ├── 사고 유형 / 위험도 (AI 자동 산출)
        ├── 발생 일시 / 장소
        ├── 식별된 위험 요인
        ├── 조치 결과
        ├── 처리 상태 (조치 대기 / 조치 완료)
        ├── 최근 수정 시간 표시 (예: "5분 전 수정")
        ├── 수정 이력 보기 버튼
        ├── 수정 버튼
        └── 조치 완료 처리 버튼 (조치 대기 상태에만 활성화)
```

---

### 10. 사고 수정 플로우

```
내 사고 관리 → 사고 상세 진입
└── 수정 버튼 탭
    └── 사고 수정 화면
        ├── 내용 수정
        └── 저장
            ├── 수정 완료 안내 토스트
            └── 수정 시간 기록 (서버에서 저장)
                └── 사고 상세로 이동
                    └── 사고 상세 화면
                        ├── 최근 수정 시간 표시 (예: "5분 전 수정")
                        └── 수정 이력 보기 버튼
                            └── 수정 이력 목록
                                └── 각 항목: 수정 시간 / 수정자 / 변경 내용 확인 가능
```

---

### 11. 필터 기반 사고 조회

```
전체 신고 현황 / 내 사고 관리
└── 필터 버튼 탭
    └── 필터 선택 화면
        ├── 등록자 (내가 등록한 건 / 부대 전체)
        ├── 처리 상태 (조치 대기 / 조치 완료)
        ├── 사고 유형 (카테고리 선택)
        └── 기간 선택
            └── 필터 적용 버튼
                └── 필터링된 목록 표시
```

---

### 12. 보고서 생성 및 출력 (관리자)

```
신고 현황 → 보고서 탭
└── 보고서 생성 화면
    ├── 기간 선택 (월간 / 분기)
    └── 생성 버튼
        ├── [성공]
        │   └── 보고서 미리보기
        │       └── 출력 / 저장 버튼
        └── [실패]
            └── 오류 안내 토스트
                └── 보고서 생성 화면 유지
```

## 알림 / 공지

### 13. 알림 수신 및 상세 확인

```
서버: 알림 발송 (미조치 기한 초과 / 가입 승인 / 반려 등)
└── 디바이스 푸시 알림 수신
    ├── [앱 백그라운드] 푸시 알림 탭
    │   └── 앱 실행 → 알림 상세로 바로 이동
    └── [앱 포그라운드] 상단 알림 표시
        └── 상단 벨 아이콘 탭
            └── 알림 목록
                └── 알림 항목 탭
                    └── 알림 상세
                        └── 관련 사고 상세로 이동
```

---

### 14. 공지사항 조회

```
바텀 네비 → 공지사항 탭
└── 공지 목록
    └── 공지 항목 탭
        └── 공지 상세
```

---

### 15. 공지 작성 (관리자)

```
공지사항 탭
└── 작성 버튼 탭
    └── 공지 작성 화면
        ├── 제목 입력
        ├── 내용 입력
        └── 첨부파일 추가 (선택)
            └── 등록 버튼
                ├── [성공]
                │   └── 등록 완료 안내 토스트
                │       └── 공지 목록으로 이동
                └── [실패]
                    └── 오류 안내 토스트
                        └── 공지 작성 화면 유지
```

---

### 16. 공지 수정/삭제 (관리자)

```
공지 상세
├── 수정 버튼 탭
│   └── 공지 수정 화면
│       └── 내용 수정 후 저장
│           └── 수정 완료 안내 토스트

└── 삭제 버튼 탭
    └── 삭제 확인 다이얼로그

        │   └── 삭제 완료 안내 토스트
        │       └── 공지 목록으로 이동
        └── [취소]
            └── 공지 상세 유지
```

## 통계

### 17. 통계 그래프 조회

```
메뉴 버튼 (관리자 / 지휘관) / 홈 스크롤 (간부)
└── 통계 그래프 화면
    ├── 월별/유형별 아차사고 분석 현황
    │   └── 범례: 사고 유형 10개 카테고리
    │       (화재·폭발·추락·끼임·부딪힘·붕괴·감전·중독·온열질환·기타)
    └── 예하부대 아차사고 등록 현황
        └── 부대별 스택 막대그래프 (조치완료 / 조치대기)
            ├── 기간 선택 (이번달 / 3개월 / 6개월 / 1년)
            └── 카테고리 필터 (전체 + 10개 유형)
```

## 체계관리자 계정 관리

### 18. 관리자 계정 생성

```
체계관리자 홈 → 계정 관리
└── 관리자 계정 목록
    └── 계정 생성 버튼 탭
        └── 계정 생성 화면
            ├── 계급 / 이름 / 군번 입력
            ├── 소속 부대 입력
            ├── 권한 선택 (인사담당자 / 안전담당자)
            └── 생성 버튼
                ├── [성공]
                │   └── 생성 완료 안내 토스트
                │       └── 관리자 계정 목록으로 이동
                └── [실패]
                    └── 오류 안내 토스트
                        └── 계정 생성 화면 유지
```

---

### 19. 관리자 권한 수정

```
계정 관리 → 관리자 계정 목록
└── 계정 항목 탭
    └── 계정 상세 화면
        └── 권한 수정 버튼
            └── 권한 선택 (인사담당자 / 안전담당자)
                └── 저장 버튼
                    └── 수정 완료 안내 토스트
                        └── 계정 상세 화면으로 이동
```

---

### 20. 관리자 권한 회수 / 계정 삭제

```
계정 관리 → 계정 상세 화면
├── 권한 회수 버튼 탭
│   └── 권한 회수 확인 다이얼로그
│       ├── [확인]
│       │   └── 권한 회수 완료 안내 토스트
│       │       └── 계정 상세 화면 유지
│       └── [취소]
│           └── 계정 상세 화면 유지
└── 계정 삭제 버튼 탭
    └── 삭제 확인 다이얼로그
        ├── [확인]
        │   └── 삭제 완료 안내 토스트
        │       └── 관리자 계정 목록으로 이동
        └── [취소]
            └── 계정 상세 화면 유지
```

---

### 27. 조치 기한 초과 처리

```
서버: 조치 기한 초과 감지
└── 관리자 푸시 알림 발송
    └── 관리자 홈
        └── 미조치 건 현황 카드 강조 표시
            └── 신고 현황 → 미조치 건 추적 관리
                └── 기한 초과 목록 확인
                    └── 해당 사고 상세
                        └── 담당 간부에게 조치 촉구 알림 발송
                            └── 간부: 알림 수신
                                └── 조치 완료 처리 플로우로 이동
```
