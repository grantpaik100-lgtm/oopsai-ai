# 권한별 화면 목록 확정

**공통 (전 권한)**

- 로그인
- 회원가입
- 공지사항
- 알림
- 내 정보 (계급 / 이름 / 군번 / 소속 부대 / 권한 표시 / 푸시 알림 설정 / 비밀번호 변경 / 로그아웃 / 회원 탈퇴)
- 통계 그래프 (월별/유형별 아차사고 분석 현황, 예하부대 아차사고 등록 현황) — 체계관리자 제외

**간부**

- 홈 (빠른 메뉴 중심)
- 아차사고 등록 (사진 촬영 → 상황 입력 → AI 분석 결과 확인 및 유형/위험도 선택 → 조치 입력 → 최종 확인 → 등록 완료)
- 아차사고 관리 (내 등록 목록 / 상세 / 수정 — 언제든 수정 가능 / 조치 완료 처리)
- 유사 사례 조회

**관리자 (인사담당자·안전담당자)**

- 홈 (신고 현황 / 미조치 건 중심)
- 아차사고 등록 (간부와 동일한 플로우)
- 아차사고 관리 (내 등록 목록 / 상세 / 수정 — 언제든 수정 가능 / 조치 완료 처리)
- 유사 사례 조회
- 전체 신고 현황 (내가 등록한 건 / 부대 전체 등록 건 필터 구분)
- 통계 분석
- 미조치 건 추적 관리
- 보고서 생성 및 출력
- 간부 계정 승인 및 권한 관리
- 게시판 관리 (공지사항 업로드)


**지휘관 (대대장급 이상)**

- 홈 (리스크 평가 → 중점 관리 → 타 부대 사례 → 의사결정 지원 스크롤 흐름)
- 사례 조회 (유사 사례 검색 및 필터링)

**체계관리자**

- 홈 (관리자 계정 현황)
- 관리자 계정 생성·수정·삭제
- 권한 부여 및 회수
