# 화면 구조도 작성(Screen Architecture)
## 공통

```
로그인
└── 회원가입

상단 공통 (전 권한)
├── 알림 (벨 아이콘)
│   └── 알림 목록
│       └── 알림 상세
└── 메뉴 버튼 (햄버거)
    └── 통계 그래프 — 관리자 / 지휘관 한정 (체계관리자 제외, 간부는 홈에서 직접 확인)
        ├── 월별/유형별 아차사고 분석 현황
        └── 예하부대 아차사고 등록 현황
```

---

## 간부

```
홈 (스크롤)
├── 내 사고 현황 요약 카드
│   ├── 전체 등록 건수
│   ├── 조치 완료
│   └── 조치 대기 (강조 표시)
├── 최근 등록 사고 미리보기 (최근 1~2건)
│   └── 탭 시 사고 상세로 이동
├── 빠른 메뉴
│   ├── 아차사고 등록
│   ├── 내 사고 관리
└── 통계 그래프
    ├── 월별/유형별 아차사고 분석 현황
    └── 예하부대 아차사고 등록 현황

사고 등록
├── 사진 촬영 (카메라 직접 실행 / 첨부 즉시 백그라운드 AI 분석 시작)
├── 상황 입력 (일시 / 장소)
├── AI 분석 결과 확인
│   ├── 사고 유형 분류 결과 (수정 가능)
│   ├── 위험도 분석 결과 (수정 가능)
│   ├── 예방 대책 제안 텍스트
│   └── 예상 결과 이미지
├── 조치 입력
│   └── 유사 사례 조회
│       └── 사례 상세 보기
├── 최종 확인
└── 등록 완료
    └── 조치 대기 / 즉시 조치 완료 선택

내 사고 관리
├── 내 등록 목록
└── 사고 상세
    ├── 수정
    └── 조치 완료 처리
        └── 조치 완료 사진 촬영 및 업로드

공지사항
└── 공지 상세

내 정보
├── 계급 / 이름 / 군번 / 소속 부대 / 권한
├── 비밀번호 변경
├── 푸시 알림 설정
├── 로그아웃
└── 회원 탈
```

## 관리자 (인사담당자·안전담당자)

```
홈 (스크롤)
├── 신고 현황 요약 카드
│   ├── 전체 등록 건수
│   ├── 조치 완료
│   └── 조치 대기 (강조 표시)
├── 미조치 건 현황 (기한 초과 강조 표시)
└── 빠른 메뉴
    ├── 아차사고 등록
    ├── 전체 신고 현황
    └── 간부 계정 승인

사고 등록
└── (간부와 동일한 플로우)

신고 현황
├── 전체 신고 목록
│   ├── 필터 (내가 등록한 건 / 부대 전체 등록 건)
│   └── 사고 상세
│       ├── 수정
│       └── 조치 완료 처리
│           └── 조치 완료 사진 촬영 및 업로드
├── 통계 분석
│   └── 부대별 / 사고 유형별 시각화
├── 미조치 건 추적 관리
└── 보고서 생성 및 출력

계정 관리 (홈 빠른 메뉴)
├── 간부 가입 승인 목록
└── 승인 상세 및 처리

공지사항
├── 공지 목록
├── 공지 상세
├── 공지 작성
└── 공지 수정/삭제

내 정보
├── 계급 / 이름 / 군번 / 소속 부대 / 권한
├── 비밀번호 변경
├── 푸시 알림 설정
├── 로그아웃
└── 회원 탈퇴
```

---

## 지휘관 (대대장급 이상)

```
홈 (스크롤)
├── 리스크 평가
│   └── 부대별/유형별 고위험 부대 현황
├── 중점 관리 대상
│   └── 기간별 집중 카테고리 현황
├── 타 부대 사례
│   └── 사례 상세 보기
└── 의사결정 지원
    └── 예방 대책 데이터 기반 제공

사례 조회
├── 유사 사례 목록 (검색 / 필터링)
└── 사례 상세 보기

공지사항
└── 공지 상세

내 정보
├── 계급 / 이름 / 군번 / 소속 부대 / 권한
├── 비밀번호 변경
├── 푸시 알림 설정
├── 로그아웃
└── 회원 탈퇴
```

## 체계관리자

```
홈
└── 관리자 계정 현황

계정 관리
├── 관리자 계정 목록
├── 계정 생성
└── 계정 상세
    ├── 계정 수정
    ├── 계정 삭제
    └── 권한 부여 및 회수

공지사항
└── 공지 상세

내 정보
├── 계급 / 이름 / 군번 / 소속 부대 / 권한
├── 비밀번호 변경
├── 푸시 알림 설정
├── 로그아웃
└── 회원 탈퇴
```
