# MVP 기능 플로우

---

## 사고 등록 플로우 (간부 / 관리자 공통)

### 1단계: 상황 설명 및 사진 등록

```
앱 실행 → 사고 등록 진입
├── 사진 첨부 (선택 사항)
│   ├── 카메라 버튼 → 카메라 실행 → 촬영 완료
│   │   └── 첨부된 사진 썸네일 표시
│   │       ├── 재촬영 버튼
│   │       └── 추가 버튼
│   ├── 갤러리 버튼 → 갤러리 선택
│   │   └── 첨부된 사진 썸네일 표시
│   │       ├── 재촬영 버튼
│   │       └── 추가 버튼
│   └── 건너뛰기 → 사진 없이 진행
├── 식별 일시 입력
├── 발생 장소 입력
└── 식별된 위험 요인 텍스트 입력창
    └── "다음" 버튼
        └── [백그라운드 AI 분석 자동 시작 — 사진 + 텍스트 동시 분석]
            └── 2단계로 진입
```

---

### 2단계: AI 추가 질문 (필요 시 / 없으면 3단계로 바로 진입)

```
"몇 가지 더 여쭤볼게요"
└── 상단 진행률 바 표시 (질문 N / 전체)
    └── 질문 순차 진행
        ├── 선택형 질문
        │   ├── 선택지 버튼 탭 → 선택됨 표시
        │   └── "또는 직접 입력" 텍스트 입력창 (선택 사항)
        └── "다음 질문" 버튼
            ├── [다음 질문 있음] → 다음 질문 표시
            └── [마지막 질문 완료]
                └── 로딩 화면 ("사고 유형을 분석하고 있어요...")
                    ├── [분석 완료] → 사고 유형 선택 화면
                    │   └── 10개 카테고리를 라디오 버튼/칩 형태로 표시
                    │       1. 화재
                    │       2. 폭발
                    │       3. 떨어짐(추락)
                    │       4. 끼임(협착)
                    │       5. 부딪힘(충격)
                    │       6. 붕괴
                    │       7. 감전 ← [AI 추천] 배지 표시 (예시)
                    │       8. 중독(질식)
                    │       9. 온열질환
                    │       10. 기타
                    │       ├── 유형 선택 → "다음" 버튼 → 3단계로 진입
                    │       └── [AI 분석 실패 / 타임아웃]
                    │           └── [AI 추천] 배지 없이 10개 카테고리만 표시
                    │               └── 사용자가 직접 선택 → 3단계로 진입
                    └── [로딩 실패 / 네트워크 오류]
                        └── 오류 안내 토스트 + 재시도 버튼
                            ├── [재시도 성공] → 사고 유형 선택 화면
                            └── [재시도 실패] → AI 추천 배지 없이 사고 유형 선택 화면
    └── "나머지 질문 건너뛰기" 버튼
        └── 사고 유형 선택 화면으로 진입 (로딩 없이 바로 표시)
```

### 3단계: 예방 대책 보기 (보기 전용)

```
[AI 예방대책 생성 중]
└── 로딩 화면 ("예방 대책을 생성하고 있어요...")
    └── 생성 완료 전 "다음" 버튼 비활성화 — 완료될 때까지 대기 필수

[AI 예방대책 생성 완료]
└── 예방 대책 카드 목록 (1~5개, AI가 유동적으로 결정)
    └── 카드 1개 구성 (세트)
        ├── AI 생성 이미지 (예방 조치 가이드 시각화)
        └── 예방 대책 텍스트
    └── 스크롤로 전체 카드 확인 (입력 없음, 보기 전용)
        └── "다음" 버튼 → 4단계로 진입

[AI 예방대책 생성 실패]
└── "예방 대책 생성에 실패했어요" 안내
    ├── 재시도 버튼
    │   ├── [재시도 성공] → 예방 대책 카드 목록 표시
    │   └── [재시도 실패] → 안내 유지
    └── "건너뛰기" 버튼 → 4단계로 진입 (예방 대책 카드 없이)
```

---

### 4단계: 조치 결과

```
"조치 결과를 선택하거나 직접 입력해주세요"

├── [AI 추천 예방 대책 선택] (기본 모드)
│   └── 3단계 예방대책 카드 목록 재표시 (다중 선택 가능)
│       └── 선택된 항목 강조 표시
│           ※ "직접 입력" 버튼 탭 시 카드 선택 초기화 및 비활성화
│
├── [수기 입력] "직접 입력" 버튼 탭
│   └── 카드 선택 비활성화 + 텍스트 입력창 활성화
│       └── 자유 텍스트 작성 (AI 추천 선택과 양립 불가)
│
├── [유사 사례 참고] 버튼 탭 (선택 사항 — 위 두 모드 모두에서 가능)
│   └── 유사 사례 목록 화면으로 이동
│       └── 사례 상세 확인 후 뒤로가기 → 4단계 복귀 (입력 내용 유지)
│
├── [조치 완료 사진 업로드] (선택 사항 — 위 두 모드 모두에서 가능)
│   ├── 카메라 버튼 → 카메라 실행 → 촬영
│   ├── 갤러리 버튼 → 갤러리 선택
│   └── 복수 장 업로드 가능
│
└── 분기
    ├── [선택 또는 수기 입력 완료] → "다음" 버튼
    │   └── 조치 완료 상태로 5단계 진입
    └── "나중에 조치" 버튼
        └── 조치 대기 상태로 5단계 진입
```

---

### 5단계: 최종 보고서 및 등록

```
보고서 미리보기 전체 표시
├── 소속 / 계급 / 성명 (자동 입력)
├── 식별 일시 — 탭 시 모달로 수정 가능
├── 식별 장소 — 탭 시 모달로 수정 가능
├── 사고 유형 — 탭 시 모달로 수정 가능
├── 위험도 — AI 자동 산출, 수정 불가
├── 식별된 위험 요인 (1단계 작성 내용) — 탭 시 모달로 수정 가능
├── 조치 결과 — 탭 시 4단계로 이동하여 수정
├── 관련 사진
│   ├── 1단계 입력 사진 (있을 경우)
│   └── 조치 완료 사진 (있을 경우 / 미첨부 시 "사진 없음" 표시)
└── 처리 상태 표시
    ├── 조치 결과 입력 완료 → "조치 완료"
    └── 나중에 조치 선택 → "조치 대기"

└── "등록하기" 버튼
    └── 등록 완료
        ├── [조치 완료] → 즉시 전체 공개
        └── [조치 대기] → 비공개 저장, 조치 완료 후 공개
```

---

### 등록 후: 관리자 반려 로직

```
[조치 완료 글 공개 상태]
└── 관리자가 조치 내용 검토
    ├── [적절] → 유지
    └── [미흡 판단] → 반려 처리
        ├── 반려 사유 입력 (필수)
        ├── 해당 글 비공개 전환
        └── 간부에게 푸시 알림 발송 (반려 사유 포함)

[간부: 반려 알림 수신]
├── [홈 화면] 반려 건 배너 표시 (반려 사유 요약)
│   └── "수정하기" 버튼
│       └── 해당 사고 수정 화면으로 이동
│           └── 조치 결과 수정 후 재제출
│               └── 즉시 전체 공개
└── [알림 탭] 반려 알림 항목 탭
    └── 반려 사유 상세 확인
        └── "수정하기" 버튼
            └── 해당 사고 수정 화면으로 이동
                └── 조치 결과 수정 후 재제출
                    └── 즉시 전체 공개 (재제출 시 승인 없이 바로 공개)
```

## 간부 영역 핵심 기능

### 조치 완료 인증 플로우

```
내 사고 관리 → 사고 상세
└── 조치 완료 처리 버튼 탭 (조치 대기 상태에만 활성화)
    └── 카메라 실행 또는 갤러리 선택
        └── 조치 완료 현장 사진 촬영 / 선택
            ├── [성공] 사진 첨부 완료
            │   └── 최종 확인 화면
            │       ├── 등록 사진
            │       ├── AI 생성 사진
            │       └── 조치 완료 사진
            │           └── 조치 완료 등록 버튼
            │               └── 서버: 사고 상태 → 조치 완료 처리
            │                   └── 완료 안내 토스트
            │                       └── 내 사고 관리 목록으로 이동
            └── [취소]
                └── 사고 상세 화면 유지
```

---

### 조치 대기 건 이어서 완료하기

```
홈 화면
└── [조치 대기 건 존재 + 조치 기한 3일 이내]
    └── 상단 배너 표시
        └── "미완료 보고서 조치 기한이 임박했어요. 조치를 완료해주세요."
            ├── "지금 완료하기" 버튼
            │   └── 해당 사고 상세로 이동
            │       └── 조치 완료 인증 플로우로 이동
            └── "나중에" 버튼
                └── 홈 화면 유지

└── [조치 대기 건 존재 + 조치 기한 여유]
    └── 내 사고 현황 카드에 조치 대기 건 개수만 표시
```

---

### 내 사고 관리

```
바텀 네비 → 내 사고 관리 탭
└── 내 등록 목록
    ├── 전체 / 조치 대기 / 조치 완료 필터
    └── 사고 항목 탭
        ├── 각 항목에 최근 수정 시간 표시 (예: "5분 전 수정")
        └── 사고 상세 화면
            ├── 등록 사진
            ├── AI 생성 사진
            ├── 조치 완료 사진 (조치 완료 시)
            ├── 사고 유형 / 위험도
            ├── 발생 일시 / 장소
            ├── 식별된 위험 요인
            ├── 조치 결과
            ├── 처리 상태 (조치 대기 / 조치 완료)
            ├── 최근 수정 시간 표시
            ├── 수정 이력 보기 버튼
            ├── 수정 버튼
            └── 조치 완료 처리 버튼 (조치 대기 상태에만 활성화)
```

## 관리자 영역 핵심 기능

### 미조치 추적 플로우

```
조치 대기 상태 사고 등록됨
└── 서버: 조치 기한 타이머 시작 (관리자 설정 기간 기준)
    ├── [기한 내 조치 완료]
    │   └── 조치 완료 처리 플로우로 이동
    └── [기한 초과]
        └── 서버: 기한 초과 감지
            ├── 관리자에게 푸시 알림 발송
            │   └── 알림 탭 → 알림 상세 → 해당 사고 상세로 이동
            └── 관리자 홈
                └── 미조치 건 현황 카드 강조 표시
                    └── 신고 현황 → 미조치 건 추적 관리
                        ├── 미조치 목록 확인
                        └── 해당 사고 상세
                            └── 담당 간부에게 조치 촉구 알림 발송
```

---

### 전체 신고 현황 조회

```
관리자 홈 → 전체 신고 현황 탭
└── 전체 신고 목록
    ├── 필터 (내가 등록한 건 / 부대 전체 등록 건)
    ├── 사고 유형별 필터
    ├── 처리 상태 필터 (조치 대기 / 조치 완료)
    └── 사고 항목 탭
        └── 사고 상세 조회로 이동
            ├── 수정 버튼
            └── 조치 완료 처리 버튼 (조치 대기 상태에만 활성화)
```

---

### 간부 계정 승인

```
관리자 홈 빠른 메뉴 → 간부 계정 승인
└── 가입 승인 목록
    └── 신청 항목 탭
        └── 승인 상세 화면
            ├── 계급 / 이름 / 군번 / 소속 부대 확인
            ├── 승인 버튼
            │   └── 승인 확인 다이얼로그
            │       ├── [확인] → 승인 완료 처리
            │       │   └── 간부에게 승인 푸시 알림 발송
            │       └── [취소] → 승인 상세 화면 유지
            └── 거절 버튼
                └── 거절 사유 입력 (선택)
                    └── 거절 확인 버튼
                        └── 서버: 계정 거절 처리
                            └── 간부에게 거절 푸시 알림 발송
```

## 지휘관 영역 핵심 기능

```
지휘관 홈 (별도 조작 없이 순서대로 스크롤)
│
├── 리스크 평가 섹션
│   └── 부대별/유형별 고위험 부대 현황
│       └── 항목 탭 → 해당 부대 사고 현황 상세
│
├── 중점 관리 대상 섹션
│   └── 기간 선택 (월간 / 분기)
│       └── 집중 카테고리 현황 표시
│           └── 항목 탭 → 해당 카테고리 사고 목록 상세
│
├── 타 부대 사례 섹션
│   └── 주요 사례 목록
│       └── 사례 항목 탭 → 사례 상세 보기
│
└── 의사결정 지원 섹션
    └── 예방 대책 데이터 기반 제공
        └── 항목별 상세 확인
```
