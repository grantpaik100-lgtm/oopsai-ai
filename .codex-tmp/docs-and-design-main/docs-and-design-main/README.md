## 🗂️ 시스템 개요

**아차사고 및 위험 요인 관리 시스템**은 군 부대 내 아차사고(Near-Miss)를 체계적으로 등록·관리·분석하는 모바일 앱 기반 시스템입니다.

| 권한 | 대상 | 주요 역할 |
|------|------|-----------|
| 간부 | 일반 사용자 | 아차사고 등록 및 사례 조회 |
| 관리자 | 인사담당자·안전담당자 | 신고 현황 운영 및 계정 승인 |
| 지휘관 | 대대장급 이상 | 부대 리스크 현황 파악 및 의사결정 |
| 체계관리자 | 최상위 운영자 | 관리자 계정 생성·수정·삭제 및 권한 부여 |
